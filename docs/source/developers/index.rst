Contributing to Open Source Code
=====================================

There are many ways that you can contribute to |PsychoPy| - even if you're not a developer! See below 
for handy how-tos on open-scource contribution to |PsychoPy|.

To discuss ideas in depth, check out the dedicated `developers section of the forum <https://discourse.psychopy.org/c/dev/10>`_.

Happy Coding Folks!!

Adding a plugin
-------------------------------------

From v2023.2, |PsychoPy| has supported plugins - these are entirely separate Python modules which, 
when installed, add content to PsychoPy (such as new Components, functions and classes). See below 
for an in-depth guide on creating and publishing your own plugin:
